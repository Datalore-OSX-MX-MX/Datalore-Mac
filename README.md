## Frequently Asked Questions (FAQ)

**What makes Datalore different from alternatives?**
Datalore combines collaborative data science and analytics platform with deep macOS integration, providing cloud-based notebooks through optimized performance and native system features.

**Is Datalore compatible with Apple Silicon?**
Yes, Datalore is built as a universal binary supporting both Intel and Apple Silicon processors with optimized performance for each architecture.
